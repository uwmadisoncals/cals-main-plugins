Adds a custom URL field for post or page titles, allowing you to replace their default permalinks.

== Description ==

This plugin adds a new "Custom Title URL" field in the post/page editor, where users can enter a new URL (e.g. 'http://www.mywebsite.com') to replace the post/page's default permalink.


== Installation ==

1. Upload 'cals_custom_post_title_url' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In post/page editor, fill out the "Custom Title URL" field to replace the post/page's default permalink
