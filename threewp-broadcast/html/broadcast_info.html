Broadcast allows you to publish posts on several blogs at once, optionally linking them together so that you only have to update the original, parent post and the child posts will all be automatically updated.

To use Broadcast, edit a post, select the blogs to which you wish to broadcast to, and then publish!

Broadcast also has some online documentation and there is information about the Premium Pack that contains several plugins that help increase productivity and work together with various 3rd party plugins.
