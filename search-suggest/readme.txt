== Description ==

This plugin makes it possible to mimic search engine's search behavior, by showing related searches and spelling suggestions.

More info:

* [Search Suggest](http://yoast.com/wordpress/search-suggest/).
* Read more about how to improve your [WordPress Search experience](http://yoast.com/wordpress-search/), so you can get the most out of this plugin.
* Check out the other [Wordpress plugins](http://yoast.com/wordpress/) by the same author.

== Installation ==

Installation is easy:

* Download the plugin and unzip it.
* Copy the `search-suggest` folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* Add `related_searches()` or `spell_suggest()` to your theme's `search.php`.
